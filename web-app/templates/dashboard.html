Field Descriptions

Model: The face recognition model used by the ML service to compare faces.

Distance metric: The mathematical method used to measure how close two face embeddings are. For example, cosine distance.

Distance: The raw difference score between the uploaded face and the matched professor face. Smaller usually means more similar.

Threshold: The cutoff value used to decide whether the comparison counts as a match. If distance is below or equal to this value, it is treated as a match.

Confidence: A confidence-related value returned by the ML library. Depending on the library version, this may be approximate rather than a calibrated probability.

Similarity score: A user-friendly score derived for display. Higher means more similar. This is mainly for presentation, not a strict scientific probability.

Is match: Whether the system judged the uploaded image and professor image to be a valid match under the threshold rule.

Source face box: The detected face region in the uploaded image, usually stored as x, y, width, and height.

Target face box: The detected face region in the matched professor image, usually stored as x, y, width, and height.

Requested at: The time when that upload attempt was made.

Original filename: The original name of the uploaded image file.

Matched name: The professor name returned as the best match.
